# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please email `security@ethora.io` with:

1. Description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Suggested fix (optional)

We aim to respond within 48 hours and will keep you updated throughout the remediation process.

## Scope

- Smart contracts in `packages/contracts/src/`
- REST API in `packages/api/src/`
- SDK in `packages/sdk/src/`

Out of scope: documentation site, UI cosmetic issues.
